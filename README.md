Source code for the journal article on composite reconstruction, providing the generic decoupling theorem for abstract Hilbert-plus-Banach problems.

Article available on Arxiv [here](https://arxiv.org/abs/2510.23322).

Submitted to Inverse Problems by IOPScience.

Public code available on [GitHub](https://github.com/AdriaJ/decouple-composite).
